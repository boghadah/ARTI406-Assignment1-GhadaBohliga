# Student Admission Records - EDA
## Dataset Source
This dataset was obtained from Kaggle:
https://www.kaggle.com/datasets/zeeshier/student-admission-records
## Dataset Size
The dataset contains 157 rows and 7 columns.
## Features (Columns Description)
- Name: Name of the student
- Age: Age of the student
- Gender: Gender of the student
- Admission Test Score: Score obtained in the admission test
- High School Percentage: Student’s percentage in high school
- City: City of the student
- Admission Status: Whether the student was accepted or rejected
## Purpose of Using This Dataset
This dataset was selected to perform Exploratory Data Analysis (EDA). The goal is to clean the data, visualize key patterns, and extract
insights related to student admission decisions
